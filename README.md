# Secure Health System

A **Python-based health system** for Lesotho that manages patients, doctors, appointments, payroll, and integrates a Sesotho chatbot.

---

## Features

- Patient registration and appointment booking  
- Doctor management  
- Payroll management  
- Medical record management  
- Sesotho chatbot for patient assistance  
- User authentication and dashboards
